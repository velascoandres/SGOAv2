Software Libre vs Software Propietario vs Software de Código Abierto

Software Propietario:

Software del cual no existe una forma libre de acceso a su código fuente, el cual solo se encuentra a disposición de su desarrollador y no se permite su libre modificación, adaptación o incluso lectura por parte de terceros.

Software de Código Abierto:

Software cuyo código fuente y otros derechos que normalmente son exclusivos para quienes poseen los derechos de autor, son publicados bajo una licencia de código abierto o forman parte del dominio público. En las licencias compatibles con la Open Source Definition el propietario de los derechos de autor permite a los usuarios utilizar, cambiar y redistribuir el software, a cualquiera, para cualquier propósito, ya sea en su forma modificada o en su forma original.

Diferencias

Software Libre

Software de Código Abierto

Software Propietario

-Nos brinda la libertad de usar el programa para cualquier propósito.

-No todo software de código abierto es software libre, algunas cuentan con licencias demasiado restrictivas.

-No nos brinda la posibilidad de revisar el código fuente.

-Nos da la libertad de estudiar como funciona el programa (código fuente) y modificarlo para que se adapte a nuestras necesidades.

- En software libre los autores de las modificaciones pueden pedir que un producto basado en el suyo sea renombrado para evitar confusiones, en código abierto se puede impedir la distribución de algunos subproductos, aunque siempre se puedan publicar como parches o añadidos al original.

-Generalmente cuanta con puertas traseras (back doors) con las cuales se puede evitar las seguridades (autenticación) y acceder al sistema para realizar cambios.

-Podemos distribuir copias del programa, ayudando a nuestro prójimo.

-Utilizar software de código abierto no siempre implica libertad, sino disponibilidad a la hora de acceder al código fuente.

-No se pueden distribuir copias ya que se atenta contra la licencia del software.

-Podemos hacer mejoras al programa y hacer público estos cambios, para que la comunidad se beneficie

-Establece al aspecto técnico como prioritario dejando de lado las cuestiones éticas.

- Las actualizaciones y el soporte solo son dadas por el fabricante.
